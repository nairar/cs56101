Description

Used NodeJS to capture array data and present it in a Form table Have installed Node,MongoDB and Rockmongo cartridges for the same in Openshift and host the website in the cloud.

I am trying to create routes in NodeJS to capture array data from static array created in NodeJS server and show the response. Hit this page url to see the json formatted data that i have stored. also you can try checking the environment parameters in the app() process by appending "/env" to the cloud url.

Purpose

The purpose of the experiment is to learn more about how NodeJS routes GET requests. I have made the code very modular to ensure that errors in coding are minimized and that I have a separate routes folder that services the various "GET" requests. I am storing a JSON array in the server and accessing it using routed "GET" calls via NodeJS api's.
